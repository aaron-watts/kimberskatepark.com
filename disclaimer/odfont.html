About This Website

This website is maintained on a voluntary basis and is in no way affiliated with Wandsworth Borough Council.

This website is intended as a convenient information hub for park visitors to quickly and easily find opening hours, planned closures and upcoming events.

Any content within the Schedule section is fetched from a calendar that is maintained by the staff at Kimber Skatepark, and the creator/owner of this website accepts no responsibility for any inaccuracies, mistakes or inappropriate content contained within this section.

Data, Privacy and Cookies

This website uses no data, user data or cookies from anyone who visits this website.

Data for the live schedule is fetched from Google Calendars, and steps have been taken to ensure this process is obfuscated onto a server so that anyone using this website does not connect to any external services beyond the scope of this website.

All assets are static and hosted locally to ensure that this website depends on no external services that may collect data or use cookies.
